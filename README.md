# Exchange-Deployment-Automation-Tool


This PowerShell script installs Microsoft Exchange Server (2016/2019/SE) completely unattended and performs basic configuration automatically. It handles server preparation, Active Directory preparation, Exchange setup, and post-installation configuration in one continuous workflow.

Benefits:

Fully automated installation without manual intervention

Unattended mode with automatic answer file generation

Includes AD prep (ForestPrep, DomainPrep)

Automatic installation of prerequisites and components

Post-install configuration (coexistence, certificates, services)

Reproducible and error-reduced for enterprise deployments

Use case: Ideal for physical servers, VMs, labs, test environments, and production Exchange deployments with consistent setup.
